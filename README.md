## 🧠 How It Was Built: PM + AI Collaboration

```
PM    → "I need a 3D office floor plan, top-down view.
         External walls, 6 zones in different colors.
         Device markers with pulse animations.
         Left panel for devices, click to focus."

AI    → [Generates complete Three.js scene — 500 lines]

PM    → "Lighting too dim. Add ambient light. Lighter gray floor.
         Add CSS labels above markers showing name + status."

AI    → [Adjusts lighting, materials, adds CSS2D labels]

PM    → "Add meeting room tables. Zone colors more subtle —
         translucent overlays instead of opaque."

AI    → ✅ Done. 3 rounds. Zero human code.
```

### Three Rules for Building with AI

**1️⃣ Natural language, not coordinates**
```
❌ "Place a cube at (12.5, 0, 8.3)"
✅ "Put a blue sensor marker in the northwest corner,
    near the wall but not touching it"
```
AI does the math. You describe *what*, AI figures out *where*.

**2️⃣ Layer by layer — validate each round**
```
Round 1 → Structure  (ground, walls, zones)
Round 2 → Objects    (devices, furniture, labels)
Round 3 → Behavior   (click, focus, panels)
Round 4 → Polish     (lighting, shadows, timing)
```
Never dump everything at once. Build like LEGO.

**3️⃣ A sketch > 1000 words**
```
Draw rough floor plan → Screenshot → "Follow this layout"
```
10x faster than describing spatial relationships in text.
